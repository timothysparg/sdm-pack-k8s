initiateKubernetesDeploy

  • initiateKubernetesDeploy(k8Deploy: KubernetesDeploy, registration: KubernetesDeployRegistration): ExecuteGoal
    • Defined in lib/deploy/goal.ts:233

    If in SDM team mode, this goal executor generates and stores the Kubernetes application data for deploying an application to Kubernetes. It returns the augmented SdmGoalEvent with the Kubernetes application information in the data property and the state of the SdmGoalEvent set to "in_process". The actual deployment is done by the kubernetesDeployHandler event handler.

    It will call defaultDataSources to populate the default repository data sources if none are provided in the registration.

    If in SDM local mode, generate the Kubernetes application data and deploy the application.

    Parameters

    • k8Deploy: KubernetesDeploy

      Kubernetes deploy object

    • registration: KubernetesDeployRegistration

      Kubernetes deploy object registration

    Returns ExecuteGoal

    An ExecuteGoal result that is not really a result, but an intermediate state.
