eligibleDeployGoal

  • eligibleDeployGoal(goalEvent: SdmGoalEvent, params: KubernetesDeployParameters): Promise<boolean>
    • Defined in lib/events/kubernetesDeploy.ts:171

    Determine if SDM goal event should trigger a deployment to Kubernetes. Specifically, is the name of the goal fulfillment equal to the name of this SDM and is the goal in the "in_process" state. Since we have improved the subscription to select for all of these values, these checks should no longer be necessary.

    Parameters

    • goalEvent: SdmGoalEvent

      SDM goal event

    • params: KubernetesDeployParameters

      information about this SDM

    Returns Promise<boolean>

    true if goal is eligible, false otherwise
